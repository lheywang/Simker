# digital VHDL example

This example show a simple four bit counter, as any other digital simulation.
The testbench is here wrote in plain VHDL.

![](https://github.com/lheywang/Simker/blob/main/images/digital_vhdl/digital_vhdl.png)

## Running it ?

Just type 

```sh
make wave
```

You'll see a waveform viewer that show the output.

![](https://github.com/lheywang/Simker/blob/main/images/digital_vhdl/digital_vhdl.gif)
